搭建博客

框架选择

 先是花费时间学习了解博客，以及初步了解搭建过程。搭建一个独立的网站，在域名、空间和内容上独立自主的博客方式多种多样。
这里选择了使用 hexo + git的方式搭建。

 

搭建步骤

 

安装hexo，以及配置

（安装需要node.js）
1、打开git bash，进入到你要放你博客的文件夹下.
2、执行如下命令安装Hexo: 
 `npm install -g hexo`
3、执行init命令初始化hexo 
 `hexo init`
4、生成静态页面 
 `hexo generate`（hexo g也可以）
5、本地启动 启动本地服务，进行文章预览调试，命令： 
 `hexo server`
7、安装hexo deployer 用于更新博客： 
 `npm install hexo-deployer-git –save` 
8、这一步在准备好git仓库，修改配置文件后，再执行。上传本地生成好的静态页面： 
 `hexo deploy`(也可以是hexo d)


 配置根目录下的_config.yml文件
    deploy:
        type: git
        repo: https://github.com/faver0099/faver0099.github.io.git
        branch: master

配置git以及部署到git

1、先new repository，建一个仓库。
2、在电脑中安装git
3、检查电脑上现有的ssh key。 
    1、cd .ssh 如果显示No such file or directory。那么就重新配置。
    2、$ ssh-keygen -t rsa -C “邮件地址@youremail.com” 
        Generating public/private rsa key pair. 
        Enter file in which to save the key (/Users/your_user_directory/.ssh/id_rsa):（这里直接回车） 
        Enter passphrase (empty for no passphrase):<输入加密串>（建议直接回车设置空密码） 
        Enter same passphrase again:<再次输入加密串>（直接回车）

最后看到这样的界面，就成功设置ssh key了：

3、将生成的ssh,设置到git
.ssh 在C盘用户文件下的.ssh文件夹中
4、部署到git
执行 hexo d

美化博客、了解hexo以及主题的配置细节

1修改语言为中文
    配置文件:language: zh-CN
2修改主题
    1
        theme: jacman
        style:
            compress: true ##会自动压缩CSS文件.
    2 关闭默认的RSS 链接 微博
        注释主题 配置文件 的控件即可.
    3没有分类以及标签的问题?
        默认没有分类和标签.
            需要在source文件夹下新建categories和tags文件夹
            在博客上添加分类和标签信息，如下：
                categories: "Hexo教程" #文章分類目錄 可以省略
                tags:
                    - 技术
                    - 记录

搭建完成地址

本地地址：
http://localhost:4000/
外网地址:
https://faver0099.github.io/

新建文章

新建博客
$ hexo new [layout] <title>
指定文章的布局（layout），默认为 post，可以通过修改 _config.yml 中的 default_layout 参数来指定默认布局。
标题最好为英文标题。

新建草稿
$ hexo publish [layout] <title>
例如：
$ hexo new photo "My Gallery"

总结、展望、下一步

可以建立外网的博客，虽然是静态的页面，想想还是有点小激动的。既然可以做到静态页面的跳转，那么也可以自定义一些页面模拟网页。
当然后续还有更多可以做的事情，了解这个框架本身，插件，还有丰富博客内容等。
